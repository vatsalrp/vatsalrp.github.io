Road Puzzle

Info: 3D Puzzle Game | Unity | C#

Description: Road Puzzle is a 3D puzzle game where the goal is to create a connected path from start to finish by rotating blocks. It features dynamic and multiple paths, adding strategic depth and replayability. The game leverages a custom pathfinding algorithm to calculate valid connections and determine the success of the solution.

An optional challenge layer is introduced through collectible stars placed along the path. Players can complete the level by connecting the path, but the optimal solution requires strategic planning to collect all available stars.

Key Contributions:

  • Developed the core gameplay mechanics and implemented a dynamic pathfinding algorithm.
  • Created all game levels, ensuring balanced difficulty and progression.
  • Built the UI from provided designs, including level selection, level progression, and saving functionality using PlayerPrefs.
  • Integrated ads and set up gameplay analytics to track player engagement and performance.
