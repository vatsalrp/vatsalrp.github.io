Slime Venture

Info: 2D Platformer Game | Unity | C#

Description: Slime Venture is a 2D platformer where players control a slime navigating through challenging levels filled with obstacles and hazards. The goal is to reach the end of each level while environmental dangers.

Players can respawn instantly upon death, encouraging trial-and-error-based learning. Smooth animations and visually appealing sprite work enhance the overall experience.

Key Contributions:

  • Developed the 2D character controller, including movement, jumping, and death/respawn mechanics.
  • Designed and built all platforming levels with increasing difficulty.
  • Created character and environmental animations using Aseprite.
  • Balanced level design to provide a fair but challenging experience.
